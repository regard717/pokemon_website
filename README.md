# Pokemon Center Explorer

這是一個可部署到 GitHub Pages 的靜態網站範例。

內容包含：

- 首頁地圖與可點擊的 Pokémon Center logo
- Pokémon Center MEGA TOKYO 的介紹頁
- 店鋪介紹文、營業時間、地址
- Google 地圖嵌入
- 限定商品展示區

## 本機開啟

直接用瀏覽器打開 `index.html` 即可，或使用任何靜態伺服器。

## 部署到 GitHub Pages

1. 建立 GitHub repository
2. 將此資料夾推上 GitHub
3. 在 repository 的 `Settings > Pages` 選擇從預設分支部署根目錄
